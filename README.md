# 💊 AI Medicine Information Chatbot

A simple web application that allows users to ask questions about medicines and get basic information like uses, dosage, and side effects.

---

## 🚀 Features

* Ask questions about medicines
* Get information like uses, dosage, and side effects
* Safe medical disclaimer in every response
* Simple chatbot interface

---

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript
* Base44
